Future Features and Version History

Some of the features all ready in the works for future versions of RB Manager:

  • The ability to lookup suggested translations using imported dictionaries
  • Multiple selections across resource tables
  • Editing subtrees within the resource bundle tree
  • The ability to print various views of resources
  • A character table for selecting characters not common to keyboards
  • ...and of course all of the bug fixes

Features added in the latest version (0.6) of RB Manager:

  • The program window now has an icon
  • Integrate the images and resources into the jar files
  • Source code made publicly available
  • Allow exporting of ICU4C resource bundles
  • ... and some bug fixes

Features added in version 0.4a of RB Manager:

  • Recently accessed resource bundles appear in the File menu for easy opening
  • New icons for the tree view
  • Ability to save, hide, or delete specific resource files without affecting the resource bundle as a whole
  • Exporting to Java class files allows for specification of the package name as well as the ability to specify public or protected objects
  • Translations can now easily accomidate multiple line text
  • RB Manager is no longer restriced to resource bundles that end in '.properties' in lower-case
  • ... and some bug fixes

Features added in version 0.3a of RB Manager:

  • The ability to merge property files and other resource bundles with the open resource bundle
  • Exporting Java compilable files (ListResourceBundle extension files)
  • Import and export support for TMX formatted XML translation files
  • The ability to search resource bundles and language files for particular translations
  • Keyboard shortcuts for the most common functions
  • A file scanning extension to the reporting tool
  • ... and some bug fixes
