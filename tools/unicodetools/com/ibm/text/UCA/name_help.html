Instructions

The Name charts provide an index to Unicode names. Each word in each Unicode character name is extracted, and used as an index for the characters.

To properly view these charts, your browser should be reasonably recent so it handles Unicode and cascading style sheets, and you should install a Unicode font and configure your browser to use it.

Notes:

  • To keep the charts from becoming too large, a 'stop-list' of words are omitted. These are:
    • AND, CAPITAL, CHARACTER, COMPATIBILITY, LETTER, SMALL, WITH
    • All script names
    • All words containing a digit
    • All Hangul Syllables
  • Unlike some of the other charts, tool-tips to reveal the names are not included (for compactness). However, if you want to know the name of any particular characters:
    • Copy the character from the cell.
    • Go to http://oss.software.ibm.com/cgi-bin/icu/tr
    • Paste in under Input 1
    • Select Output 1: Any - Name
